# Brasa Balance — Contexto do Projeto

## Glossário

| Termo | Significado |
|-------|-------------|
| **Sugestões da Staff** | Página separada que exibe o conteúdo de `docs/sugestoes-staff.md` em formato visual para jogadores |
| **Página separada** | Tela acessada via link no header, sem roteador (state `page: "pvp" | "pve" | "staff"`) |
| **Tabela resumo** | Tabela com colunas Classe, HP Full VIG 3★, Dano Full VIG 3★, Ação, Prioridade |
| **Prioridade** | Nível da sugestão: 🟡 Média, 🟢 Baixa, — (nenhuma) |
| **Full VIG 3★** | Configuração padrão de comparação entre classes: todos os pontos em Vitalidade, arma 3 estrelas |
| **Markdown como fonte** | `docs/sugestoes-staff.md` é o único ponto de verdade; a página o parseia para JSX |
